## Why On-Device AI?

Running AI models locally offers:

- Lower latency and faster response times
- Enhanced privacy and data security
- Flexibility for edge and offline scenarios

Learn how to deploy and optimize models for on-device inference using Foundry Local and ONNX Runtime.
